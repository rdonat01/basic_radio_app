### Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/rdonat01/basic_radio_app.git
   ```

2. Navigate to the project directory:
    ```bash 
    cd radio_app
    ```
3. Install dependencies:
    ```
    flutter pub get
    ```

4. Run the app:
    ```
    flutter run
    ```
